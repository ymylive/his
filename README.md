# 轻量级 HIS 课程设计

本仓库用于存放“轻量级医院信息管理系统（HIS）”课程设计文档与后续代码实现材料。项目目标是完成一个适用于小型医院场景的门诊 + 基础住院系统，并满足课程对功能、代码质量、测试数据和答辩展示的要求。

## 课题定位

系统覆盖以下核心业务范围：

- 患者信息管理
- 医生与科室管理
- 医疗记录管理
- 病房与床位管理
- 药房与药品管理
- 基础统计报表
- 基础住院流程

核心业务闭环包括：

1. 患者建档
2. 门诊挂号
3. 医生接诊与病历录入
4. 处方开立与药房发药
5. 入院登记与床位分配
6. 住院状态维护
7. 出院登记与床位释放

## 硬性实现约束

根据题目要求，项目实现需遵循以下约束：

- 所有源码统一使用 `C` 语言
- 核心数据组织采用链表实现
- 数据持久化使用 `txt` 文本文件
- 源码需包含必要注释，便于代码检查
- 需要准备可用于现场评测的原始数据文件
