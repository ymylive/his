# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [2.0.0] - 2026-03-29

### Added - 全面界面可视化增强

#### 平台支持
- **macOS 支持** - 新增 macOS (Intel 和 Apple Silicon) 构建和打包脚本
- **多平台 Release** - GitHub Actions 自动构建 Windows 和 macOS 版本

#### 新增通用 UI 组件
- **表单标签组件** (`Workbench_draw_form_label`) - 支持必填字段标识（红色星号）
- **状态徽章组件** (`Workbench_draw_status_badge`) - 彩色背景的状态指示器
- **床位网格组件** (`Workbench_draw_bed_grid`) - 可视化床位状态展示（绿色=可用，红色=占用）
- **数据表格组件** (`Workbench_draw_data_table_header`, `Workbench_draw_data_table_row`) - 统一的表格展示
- **图标按钮组件** (`Workbench_draw_icon_button`) - 带图标和标签的增强按钮
- **进度条组件** (`Workbench_draw_progress_bar`) - 显示库存水平、完成度等
- **搜索框组件** (`Workbench_draw_search_box`) - 集成输入框和搜索按钮

#### 管理员工作台改进
- 患者管理：使用搜索框组件，列表项选中高亮，状态徽章显示住院状态
- 科室医生维护：表单字段必填标识，分组标题，优化布局对齐

#### 挂号员工作台改进
- 患者建档：所有表单字段使用标签组件，必填字段清晰标识，分组展示
- 患者查询：统一搜索体验，结果列表选中高亮
- 挂号办理：状态徽章显示挂号状态，优化流程指引
- 挂号查询：改进查询功能，优化记录列表展示

#### 医生工作台改进
- 待诊列表：状态徽章显示患者状态，列表项选中高亮
- 患者历史：搜索框查询历史，时间线展示
- 接诊录入：表单必填字段标识，分组展示（基本信息、诊断信息、处理方案）
- 检查管理：状态徽章显示检查状态，优化列表展示

#### 患者自助工作台改进
- 服务首页：状态徽章显示各类状态，卡片式信息展示
- 我的挂号：状态徽章显示挂号状态（待诊/已完成/已取消）
- 我的看诊：时间线展示看诊记录，状态徽章显示就诊状态
- 我的住院：卡片展示住院信息，状态徽章显示住院状态
- 我的发药：优化发药记录列表，状态徽章显示发药状态

#### 住院登记工作台改进
- 入院登记：表单必填字段标识，分组展示（患者信息、床位分配、入院详情）
- 出院办理：4 步流程指引，分组展示出院流程和信息
- 住院查询：搜索框改进查询，状态徽章显示住院状态（住院中/已出院）

#### 病区管理工作台改进
- 病房总览：**床位网格可视化**，左侧病房列表，右侧床位网格（绿色/红色）
- 床位状态：表单标签改进，状态说明图例，状态徽章显示
- 转床调度：必填字段标识，3 步操作指引，左右分栏布局
- 出院检查：流程说明，检查项目清单

#### 药房工作台改进
- 药品建档：必填字段标识，分组展示（基本信息、库存信息、价格信息）
- 库存查询：搜索框改进查询，**进度条显示库存水平**，状态徽章分级（充足/低库存/缺货）
- 发药处理：表单改进，流程步骤指引
- 低库存预警：状态徽章醒目显示，进度条显示库存百分比，按紧急程度排序

### Changed
- 所有角色工作台统一使用新的 UI 组件
- 表单布局优化，增加视觉层次和间距
- 搜索功能统一为搜索框组件
- 状态信息统一使用彩色徽章展示

### Improved
- 用户体验：操作流程更清晰，视觉反馈更及时
- 代码质量：组件复用，减少重复代码
- 可维护性：统一风格，模块化设计
- 可视化程度：床位网格、进度条、状态徽章等直观展示

## [1.0.0] - 2026-03-XX

### Added
- 七角色专门桌面工作台
- 中文字体兜底与 glyph seed 扩展
- 响应式布局 helper
- 患者自助挂号、挂号员接待、医生接诊、住院/病区、药房、管理员跨角色闭环
- 内置答辩型模拟数据
- 28/28 测试全绿

### Features
- 患者工作台：自助挂号、病历/检查摘要、住院状态、发药记录、药品说明
- 挂号员工作台：患者建档与修改、患者查询、挂号办理、挂号查询与取消
- 医生工作台：待诊、患者历史、接诊录入、检查管理、visit handoff 交接
- 住院登记员工作台：入院、出院、住院查询
- 病区管理员工作台：病房总览、床位状态、转床、出院前检查
- 药房工作台：药品建档、补货、发药、库存查询、低库存预警
- 管理员工作台：患者总览、科室/医生维护、病房药品总览、系统信息
